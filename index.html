Game Rules [X]

Game Types

Solid Color: The classic match mode. Use sliders to match the reference.

Luminance Match (Brightness): Look past the Hue! You must output a color that matches the exact perceived mathematical brightness (grayscale value) of the target.

Color Sort (Unscramble): The sliders vanish. Tap the scrambled blocks to swap them and rebuild the perfect mathematical gradient.

Hex-dle (Decoder): 6 tries to guess the exact 6-character Hex Code (Wordle rules).

Haystack Acuity: Grid vision test. Find the odd one out.

Color Telephone: Genetic Drift. Your output becomes the next round's target.

The Mixer (Midpoint): Find the exact mathematical 50% midpoint between Color A and Color B.

Gradient Matcher: 2 Colors + Angle. The next dimension of matching.

Palette Architect: Base color is locked. Create exact harmonious schemes.

Studio Mode: Pure utility. No timer. Lock In exports your CSS.

Modifiers & Objectives

Darkroom Mode: Heavy interference. The app randomly flickers and dims, disrupting your perception of brightness.

WCAG Contrast: Achieve 4.5:1 (AA) minimum text contrast against the target.

Complementary: Find the exact mathematical opposite of the base color.

Visual Map: Replaces sliders with a 2D color picker spectrum.

Blind Hardcore: No visual reference. Rely entirely on the math code.

Terminal Mode: No UI. Type raw CSS code from memory to match.

Memory Mode: Target is visible for 3 seconds, then vanishes. Recall and recreate.

Illusion Framing: Triggers the Albers Effect using violently contrasting borders.

Vision Simulator: Apply medical colorblindness filters over the entire game.

Features

Daily Rush: 60s survival. The sequence of colors is seeded globally every day.

Ghost Replays (Duels): When sharing Duel links, your slider movements are replayed to your opponent.

CMYK Mode: Subtractive ink mixing. Red + Green no longer makes Yellow.
